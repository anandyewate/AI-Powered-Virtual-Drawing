# 🎨 Air Canvas using MediaPipe

An AI-powered virtual drawing application that allows users to draw in the air using hand gestures. Built using MediaPipe and OpenCV, this project demonstrates real-time hand tracking and gesture-based interaction.

---

## 🚀 Features

- ✋ Real-time hand tracking (21 landmarks)
- ☝️ Draw using index finger
- ✌️ Gesture-based color selection
- 🎚️ Dynamic brush size using finger distance
- 🎨 Dark UI with smooth drawing
- 🧽 Eraser functionality
- 💾 Save drawings
- ⚡ Optimized for low latency

---

## 🧠 Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
